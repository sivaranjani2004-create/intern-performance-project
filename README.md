# Intern Performance Prediction System

## Project Overview
Machine learning model to predict intern performance based on task completion, consistency, and engagement metrics.

## Tech Stack
- Python, Flask, Scikit-learn, Pandas, NumPy

## Model Performance
| Algorithm | Accuracy |
|-----------|----------|
| Decision Tree | 98% |
| Random Forest | 98% |
| **Best Model** | **Random Forest** |

## API Endpoint
- URL: http://127.0.0.1:5000/predict
- Method: POST
- Input: JSON with 9 features
- Output: High / Medium / Low

## How to Run
```bash
python train_model.py
python app.py
```

## Features Used
1. Tasks Assigned
2. Tasks Completed
3. Avg Task Time
4. Attendance %
5. Consistency Score
6. Engagement Score
7. Feedback Score
8. Learning Progress
9. Deadline Misses
